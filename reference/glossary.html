R4

医学用語集

本サイト中で噛み砕いた用語を、五十音順(+カタカナ語)にまとめた。各用語には出現ページへのリンクがあるので、読み進めながら辞書として参照できる。

50+ terms カフェイン 心血管 代謝

薬理・中枢神経系

アデノシン / アデノシン受容体拮抗

アデノシンはATPの代謝産物で、脳内で眠気と血管拡張を引き起こす神経伝達物質的な分子。カフェインは構造が似ており、アデノシン受容体(A1, A2a)に「代わりに座る」ことでシグナルをブロックする。覚醒はアデノシンの仕事を奪った結果として生じる。

関連: ADD.02 脳血流26%低下

カフェイン(Caffeine)

中枢神経刺激薬。成人上限は400mg/日、12〜17歳は100mg/日、12歳未満はゼロを推奨(FDA)。半減期は CYP1A2 型により3〜10時間で個人差が大きい。

関連: CH.01、ADD.01

カフェイン離脱症候群

慢性摂取者が中止・減量時に呈する症状群(頭痛、疲労感、抑うつ、集中困難、イライラ、悪心、筋肉痛)。DSM-5、ICD-11で正式に疾患分類。摂取中止から12〜24時間で発現、2〜9日で寛解。

関連: CH.11 依存と離脱

遺伝学・薬理遺伝

CYP1A2 / rs762551

肝臓の薬物代謝酵素。カフェインの95%以上を分解する。rs762551 SNP(一塩基多型)により AA型(速代謝)、AC/CC型(遅代謝)に分かれる。遅代謝者は大量カフェイン摂取で心筋梗塞リスクが上昇する。

関連: ADD.01

SNP(一塩基多型)

DNAを構成する4種の塩基(A, T, G, C)のうち、ある一か所が個人によって異なる、ごく小さな遺伝子のゆらぎ。ヒトゲノム全体で1000万以上存在する。薬物代謝や疾患リスクに関連するSNPが近年、個別化医療で活用されている。

循環器・心血管

QTc / トルサード・ド・ポワント

QTc は心電図上で心室が電気的に再分極するのに要する時間(心拍数で補正)。延長すると「トルサード・ド・ポワント」と呼ばれる多形性心室頻拍を誘発し、心室細動 → 心停止へ移行しやすい。Mandato 2025 レビューでは、ED摂取でQTc延長が66.7%の研究で観察された。

関連: CH.02

血管内皮機能

血管内側の「内皮細胞」による一酸化窒素(NO)産生・炎症抑制・抗血栓作用。エナジードリンクの常用は内皮機能を障害し、血管をけいれんしやすく、詰まりやすくする。若年者の突然の狭心症・心筋梗塞の基盤。

ウィドウメーカー病変

左前下行枝(LAD)近位部の重度狭窄・閉塞。「未亡人製造機」と呼ばれるほど致死率が高く、心臓前壁の広範な心筋を養う血管。トリニダード系24歳男性の症例ではEDの短時間大量摂取後にこの病変がプラーク破裂を起こした。

関連: CH.02

高血圧クリーゼ

血圧180/120 mmHg以上の緊急事態。英国ノッティンガムの50代男性は254/150 mmHg でこれを凌駕する異常値だった。臓器障害(脳卒中、解離、心不全、腎障害)を伴うと「高血圧緊急症」と呼ばれる。

腎・内臓

急性腎障害(AKI, Acute Kidney Injury)

数時間〜数日で腎機能が急速低下。KDIGOでステージ1-3に分類。ステージ3は血清Creが3倍以上もしくは尿量ほぼゼロの状態。ED誘発AKIは脱水・ADH上昇・糖負荷の3機序で起こる。

関連: ADD.07

横紋筋融解症(Rhabdomyolysis)

骨格筋細胞が壊れ、ミオグロビンが大量に血中に流出する病態。ミオグロビンが腎糸球体を詰まらせてAKIを起こし、カリウム急上昇で致死性不整脈を誘発する。運動+脱水+ED が最悪の組み合わせ。

トキシック・トライアド症候群

胃炎・肝炎・膵炎の三重同時発症。2019年の American Journal of the Medical Sciences 掲載の46歳男性症例(1日2-3本のモンスター × 4ヶ月)が古典例。ED中止で可逆。

関連: CH.05

バソプレシン(ADH)

抗利尿ホルモン。腎集合管で水再吸収を促進。激しい運動や脱水で分泌が増え、腎血流を再分配するため、運動+ED で腎髄質虚血のリスクが高まる。

代謝・細胞シグナル

mTORC1 / RAG GTPase

mTORC1は細胞の成長・増殖の中央司令塔。アミノ酸・エネルギー・成長因子を統合し、タンパク質合成と代謝リプログラミングを号令する。RAG GTPaseはアミノ酸センサーからの情報でmTORC1をリソソーム表面に運び活性化する。白血病ではタウリン→TAUT→RAG→mTORC1→解糖系の経路で燃料になる。

関連: ADD.03

解糖系(Glycolysis)

グルコース1分子を2分子のピルビン酸に分解しATPを素早く産生する代謝経路。がん細胞はこれを過剰に回す(ワールブルク効果)。白血病の爆発的増殖のエネルギー源。

インスリン抵抗性 / シュガークラッシュ

インスリンの効き目が鈍くなる状態(代謝症候群・2型糖尿病の入口)。シュガークラッシュは糖の急速摂取によるインスリン急上昇 → 血糖急降下 → 脱力・眠気の循環。

関連: CH.06

白血病・腫瘍学

タウリン / SLC6A6 / TauT

タウリン = 2-アミノエタンスルホン酸。SLC6A6遺伝子がタウリン輸送体TauTをコードし、白血病幹細胞が骨髄のタウリンを吸い上げる。SLC6A6高発現AMLは予後不良かつベネトクラクス耐性。

関連: CH.07、ADD.03

AML / CML / MDS
  • AML:急性骨髄性白血病
  • CML:慢性骨髄性白血病
  • MDS:骨髄異形成症候群(AML前段階)

いずれも骨髄の造血幹細胞から発生する悪性疾患。Nature 2025 論文でタウリンとの関連が示された。

ベネトクラクス(Venetoclax)

BCL-2 阻害薬で、AML治療の主力分子標的薬の一つ。SLC6A6高発現サブタイプのAMLではベネトクラクス耐性を示すことが Nature 2025 で示された。

内分泌

HPT軸 / HPG軸

視床下部(H)-下垂体(P)-甲状腺(T)/ 性腺(G)の制御ループ。TSH→T4/T3(HPT)、FSH/LH→性腺ホルモン(HPG)。2026年1月公開のWistarラット研究で、EDの4ヶ月投与が両軸を乱した。

関連: ADD.05

メラトニン

松果体から夜に分泌される睡眠・抗酸化ホルモン。夜間の人工光はこれを抑制する。メラトニンは骨芽細胞分化も助けるため、夜勤+高カフェインは骨代謝にも悪影響。

脳・眼・皮膚

ipRGC / 視交叉上核(SCN)

ipRGC = 内因性光感受性網膜神経節細胞。メラノプシンで青色光を感知し、SCNに明るさ情報を送る。SCNは体内時計の中枢。午前中の屋外光曝露で体内時計が整う。

眼圧(IOP)/ 緑内障

眼圧の正常範囲10-21 mmHg。緑内障は視神経が眼圧で障害され視野欠損する進行性疾患。カフェインは急性にIOPを1-4mmHg上昇させる。

関連: ADD.06

ケラチノサイト / 皮脂細胞

表皮の主要細胞(ケラチノサイト)と皮脂腺細胞(sebocyte)。高グリセミック負荷→インスリン→mTORC1→ケラチノサイト増殖/皮脂腺活性化でニキビ悪化。

ディスバイオシス / 腸内細菌叢

腸内細菌叢(gut microbiota)の善玉・悪玉のバランス崩壊。人工甘味料がこれを乱し、全身炎症・代謝撹乱・皮膚疾患・メンタルヘルスに影響する可能性。

公衆衛生・疫学・マーケティング

AmED(Alcohol mixed with Energy Drinks)

アルコールとエナジードリンクの混合摂取。Wide-awake drunkenness(目の醒めた泥酔)を起こし、ビンジドリンキング・飲酒運転・性的リスク行動を増やす公衆衛生上の危険行動。

関連: CH.10

Wide-awake drunkenness

カフェインの覚醒がアルコールの鎮静をマスクし、泥酔しているのに本人は気づかない状態。飲酒量増加と事故リスクの基盤。

中心経路 / 周辺経路(ELM)

精緻化見込みモデルの2経路。中心経路=内容の熟考、周辺経路=感情・有名人・音楽など。eスポーツ広告は周辺経路100%、中心経路情報37%、水分・カフェイン量言及0%。

関連: ADD.09

外部不経済(negative externality)

ある経済活動が、取引当事者でない第三者や将来世代に押し付ける費用。ED産業248億ドルの陰で将来の医療費が集計されないまま溜まる構造。

ピアプレッシャー / 社会資本

仲間からの同調圧力 / 人間関係で得られる評判。子ども消費社会では、人気ブランドの缶を持つこと自体が社会資本として機能する。

検査・方法論

ASL / multi-TE ASL MRI

Arterial Spin Labeling。血液の水分子スピンを磁場で標識し、造影剤なしで脳血流を計測するMRI技術。multi-TE ASLはエコー時間を変えて血液脳関門水交換も定量化できる。

コメットアッセイ(Comet assay)

単細胞電気泳動法。DNA鎖切断があると、細胞内DNA断片が彗星状に泳動される。新生児リンパ球での陽性所見は発達段階のDNA損傷を示唆する。

オッズ比 / 相対リスク

症例対照研究などで曝露群の結果オッズを非曝露群と比較した値。1を超えると曝露が結果を増やす方向に関連。NHS3では妊娠高血圧のOR 1.60(95%CI 1.12-2.29)。

系統的レビュー / メタアナリシス

あるテーマの個別研究を体系的に収集・評価する手法。Mandato 2025(37研究統合)、Newcastle 大学 2024(57研究、21か国、120万人)などが本サイトの主要な証拠群。
